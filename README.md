# 前言

毫无疑问的是：区块链是一场革命。 你可以拥抱它，也可以让道与它。除此之外，恐怕你别无选择。 

我们强烈建议九零之后的年轻人积极拥抱它，因为“区块链时代”是属于你们的时代，当仁不让，应该是你们的态度——其实也是你们应该担负起来的职责！
